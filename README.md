**AutoGLM Android 纯血版**！🚀

基于AutoGLM 改造的纯血 Android 版，源码直接翻译，将adb控制指令改为直接调用Android无障碍服务，无需连接电脑，无需配置ADB，无需跑Python脚本。



---

## 🌟 为什么选择这个版本？(对比原版 AutoGLM)

### ❌ 原版 AutoGLM 方案：
*   💻 **依赖电脑**：必须时刻连接电脑才能运行，不仅麻烦还限制了使用场景。
*   🐍 **环境复杂**：需要安装 Python、配置 ADB、处理各种依赖库...小白直接劝退！😭
*   🔌 **连接不稳定**：数据线/无线调试经常断连，动不动就报错。
*   🐢 **链路过长**：电脑截图 -> 传输 -> 识别 -> 发送指令，这一套下来延迟感人。

### ✅ 本项目的纯血 Android 方案：
*   📱 **完全独立**：**安装一个 APP 就搞定！** 手机就是大脑，随时随地都能跑！🏃‍♂️
*   ⚡️ **零配置**：不用配环境！不用敲代码！开箱即用，简直是懒人福音！🎉
*   🖐️ **原生操控**：直接调用 Android 无障碍服务，模拟手指点击/滑动，操作丝般顺滑~
*   🗣️ **语音交互**：内置语音识别功能，动动嘴就能指挥手机干活，真正的“贾维斯”体验！🎙️
*   👀 **实时反馈**：贴心的**悬浮窗**设计，AI 现在的状态、在想什么、要做什么，一目了然！

## 🚀 食用指南 (只需 3 步)

### 1️⃣ 第一步：赋予“超能力” 🦸‍♂️
首次打开 APP，系统会提示开启两个关键权限，别犹豫，开它！
*   **无障碍服务** (Accessibility Service)：让 AI 拥有替你点击屏幕的手指 👆
*   **悬浮窗权限** (Overlay Permission)：让 AI 拥有展示状态的小窗口 💬

### 2️⃣ 第二步：配置“大脑” 🧠
本应用需要“智脑”支持（API Key）：
*   去 [智谱AI开放平台](https://bigmodel.cn/usercenter/proj-mgmt/apikeys) 领一个 API Key (新用户通常有免费额度哦！💰)。
*   在 APP 点击右上角设置，填进去，保存！✅

### 3️⃣ 第三步：见证奇迹 ✨
*   **文字指令**：输入“打开抖音搜索喜德胡AI编程” 🔍
*   **语音指令**：按住话筒说话，松开即发！
*   点击发送后，**请放开双手**，静静看着它表演！😎
